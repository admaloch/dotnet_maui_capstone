# WGU Cloud Planner Android App

<h2>Project Overview</h2> <p>A cloud-synced mobile application built with <strong>C# and .NET MAUI</strong> to help students manage their academic degree plans across multiple devices. This capstone project addresses the challenge of maintaining consistent degree progress tracking when switching between devices, providing automatic multi-device synchronization through <strong>Google Firebase</strong> (Authentication & Firestore) to ensure academic data is always accessible and protected.</p><h2>Key Features</h2> <ul> <li><strong>Cross-Platform:</strong> Built with .NET MAUI for Android with future iOS potential</li> <li><strong>Cloud Sync:</strong> Real-time data synchronization across devices via Firebase Firestore</li> <li><strong>Academic Planning:</strong> Manage terms, courses, assessments, and progress tracking</li> <li><strong>Offline Capability:</strong> Full functionality without network connection</li> <li><strong>Secure Authentication:</strong> User isolation and data security through Firebase Auth</li> </ul><h2>Technical Implementation</h2> <ul> <li><strong>Frontend:</strong> .NET MAUI, C#, XAML</li> <li><strong>Backend:</strong> Google Firebase (Authentication, Firestore)</li> <li><strong>Architecture:</strong> MVVM Pattern</li> <li><strong>Local Storage:</strong> SQLite with cloud migration layer</li> </ul><h2>Project Focus & Documentation</h2> <p>This project emphasized comprehensive <strong>software engineering practices</strong> beyond just coding, including detailed documentation of:</p> <ul> <li><strong>System Design & Architecture</strong></li> <li><strong>SDLC Methodology</strong> (Adapted Agile with 3 focused sprints)</li> <li><strong>Quality Assurance</strong> and rigorous test planning</li> <li><strong>Deployment Strategy</strong> with phased rollout</li> </ul><h2>Project Status</h2> <ul> <li>✅ Core development complete</li> <li>✅ Firebase integration implemented</li> <li>✅ Comprehensive
